# Spring Boot RESTful Service

#### APIs:
- GET /todos         (Get all todo list)<br>
- GET /todos/{id}    (Get todo by ID)<br>
- POST /todos        (Create new todo)<br>
- PUT /todos/{id}    (Update todo by ID)<br>
- DELETE /todos/{id} (Delete todo by ID)<br>

#### Run Command
1. `mvn install`
2. `mvn spring-boot:run` or `./mvnw spring-boot:run`
